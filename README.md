# OVARALL E-Commerce Website

A complete e-commerce website built with PHP and MySQL.

## Features
- Product listing with categories and subcategories
- Shopping cart with AJAX quantity updates
- User authentication (login/register)
- Admin dashboard for managing products, orders, and users
- Responsive design with Bootstrap 5

## Technologies Used
- PHP 7.4+
- MySQL
- Bootstrap 5
- JavaScript (AJAX)
- HTML5/CSS3

## Installation
1. Clone the repository
2. Import `database.sql` to MySQL
3. Copy `includes/config.example.php` to `includes/config.php` and update database credentials
4. Run on XAMPP/WAMP server
